# academica-alumnos

Académica-alumno es un programa elaborado con Windows Forms utilizando lenguaje programación c# y Base De Datos en SQL server.

Permite ver registros y crear estos mismos ya sea de docente materia alumnos, cuenta con botones como nuevo, guardar, modificar, DataGridViews, labels, Textbox y una barra de busqueda.

Todo esto visto en las clases de programación l y utilizando el mismo código, sin embargo, me dió problemas en la base de datos en la conexión, y la creación de registros, se buscaron soluciones externas, sin embargo, el problema persiste, por tal suceso la aplicación no está terminada al 100 %.
